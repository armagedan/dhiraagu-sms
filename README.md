# Dhiraagu SMS
PHP SDK for [Dhiraagu Bulk SMS Gateway](https://bulkmessage.dhiraagu.com.mv)

## Installation
You can install **dhiraagu-sms** via composer or by downloading the source.

**Via Composer:**

```
composer install dash8x/dhiraagu-sms
```
